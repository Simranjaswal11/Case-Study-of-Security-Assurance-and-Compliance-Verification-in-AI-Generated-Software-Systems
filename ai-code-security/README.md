# Security Assurance and Compliance Verification in AI-Generated Software Systems

**Final Year Project — Department of Computer Science and Technology, Chitkara University**

This repository contains all scripts, prompts, and analysis artifacts used in our empirical study evaluating the security of code generated by AI coding assistants (GPT-4o, GitHub Copilot, and Claude 3.5 Sonnet) against the OWASP Top 10 (2021) vulnerability taxonomy.

---

## Repository Structure

```
ai-code-security/
├── prompts/
│   └── prompt_corpus.json        # 30 task prompts across 3 domains
├── artifacts/                    # Generated code artifacts (90 total, .py files)
│   ├── gpt4o/
│   ├── copilot/
│   └── claude/
├── results/
│   ├── bandit_results.json       # Bandit scan outputs
│   ├── semgrep_results.json      # Semgrep scan outputs
│   ├── pipaudit_results.json     # pip-audit outputs
│   └── aggregated_results.csv    # Final merged results
├── scripts/
│   ├── generate_code.py          # Submit prompts to OpenAI / Anthropic APIs
│   ├── run_bandit.py             # Run Bandit on all artifacts
│   ├── run_semgrep.py            # Run Semgrep with OWASP ruleset
│   ├── run_pipaudit.py           # Run pip-audit on artifacts
│   ├── aggregate_results.py      # Merge and deduplicate findings
│   └── manual_review_template.csv # Checklist for manual review
└── requirements.txt
```

## Setup Instructions

### 1. Install dependencies

```bash
pip install -r requirements.txt
pip install bandit semgrep pip-audit openai anthropic
```

### 2. Set API keys

```bash
export OPENAI_API_KEY="your-openai-key"
export ANTHROPIC_API_KEY="your-anthropic-key"
# GitHub Copilot artifacts were collected manually via VS Code extension
```

### 3. Run the pipeline

```bash
# Step 1: Generate code from prompts (GPT-4o and Claude; Copilot manually)
python scripts/generate_code.py

# Step 2: Run SAST scans
python scripts/run_bandit.py
python scripts/run_semgrep.py
python scripts/run_pipaudit.py

# Step 3: Aggregate and deduplicate results
python scripts/aggregate_results.py
```

## Methodology Summary

- **30 task prompts** across 3 domains: user authentication, data management, data transmission
- **3 AI tools**: GPT-4o, GitHub Copilot, Claude 3.5 Sonnet → **90 total artifacts**
- **Detection pipeline**: Bandit (Python SAST) → Semgrep (OWASP ruleset) → pip-audit → Manual review (25% sample, ~22 artifacts)
- **Classification**: OWASP Top 10 (2021) + CVSS v3.1 severity tiers

## Notes

- All prompts were submitted without explicit security instructions to simulate real developer usage.
- No generated code was modified before scanning.
- GitHub Copilot artifacts were collected manually via VS Code (no public API available at time of study).
